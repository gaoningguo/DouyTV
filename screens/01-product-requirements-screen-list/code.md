# Product Requirements Document: Multi-Content Aggregator (Douyin Style)

## Project Overview
A personal-use content aggregator inspired by Douyin's immersive UI. No login or social features.

## Target Audience
Individual users looking for a single portal for diverse entertainment content.

## Content Modules
1. **Short Video**: Full-screen vertical swipe feed.
2. **Live Stream**: Grid layout for discovering active streams with an immersive viewing mode.
3. **Music**: Discovery-focused player with trending charts and category browsing.
4. **Manga/Novels**: Clean reader interface with a shelf-style discovery page.

## Key Screens to Generate
1. **Home (Short Video)**: The main immersive feed.
2. **Live Discovery**: A hub for browsing live content.
3. **Music Hub**: Trending tracks and music discovery.
4. **Manga & Novel Portal**: Combined shelf for reading material.
5. **Personal Library**: A place for bookmarks and history (no social).
